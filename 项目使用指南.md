## 安装插件

### 方式一：从磁盘安装（推荐）

1. 找到编译好的插件文件：`build/libs/daily-report-plugin-1.0.0.jar`
2. 打开IntelliJ IDEA
3. 进入设置：`File` -> `Settings`（或按 `Ctrl+Alt+S`）
4. 左侧选择 `Plugins`
5. 点击右上角的齿轮图标 ⚙️
6. 选择 `Install Plugin from Disk...`
7. 浏览并选择 `daily-report-plugin-1.0.0.jar`
8. 点击 `OK`
9. 重启IDEA

### 方式二：从Marketplace安装

如果插件已发布到JetBrains Plugin Marketplace：

1. 打开设置：`File` -> `Settings`
2. 左侧选择 `Plugins`
3. 点击 `Marketplace` 标签
4. 搜索 "Daily Report Plugin"
5. 点击 `Install` 按钮
6. 重启IDEA

## 配置插件

### 1. 打开配置界面

安装并重启IDEA后，打开配置：

- `File` -> `Settings` -> `Tools` -> `日报周报插件`
- 或使用快捷键：`Ctrl+Alt+S`，然后选择 `Tools` -> `日报周报插件`

配置界面包含三个标签页：
- **项目配置**：管理要分析的项目
- **输出配置**：设置报告输出路径
- **AI配置**：配置AI服务参数

### 2. 项目配置

#### 添加项目

1. 切换到"项目配置"标签页
2. 点击 `添加项目` 按钮
3. 在弹出的文件选择对话框中选择Git项目的根目录
4. 点击 `确定`

#### 启用/禁用项目

- 选中一个项目
- 点击 `启用` 按钮使其在报告中可见
- 点击 `禁用` 按钮将其从报告中排除

#### 删除项目

- 选中一个项目
- 点击 `删除` 按钮
- 确认删除

### 3. 输出配置

1. 切换到"输出配置"标签页
2. 点击 `浏览` 按钮选择报告输出目录
3. 默认路径：`C:\Users\你的用户名\工作报告\`
4. 点击 `应用` 保存配置

### 4. AI配置

#### 选择协议类型

支持以下协议：

| 协议类型 | 说明 | 默认Base URL |
|---------|------|-------------|
| HuaweiCloud | 华为云ModelArts服务 | https://api.modelarts-maas.com/v2/chat/completions |
| Baidu Qianfan | 百度千帆Coding Plan | https://qianfan.baidubce.com/v2/coding/chat/completions |
| OpenAI | OpenAI兼容接口 | https://api.openai.com/v1/chat/completions |
| Anthropic | Anthropic兼容接口 | https://api.anthropic.com/v1/messages |

#### 配置参数

1. **协议类型**：从下拉列表选择
2. **Base URL**：API的完整地址（选择协议后自动填充，可手动修改）
3. **API Key**：你的API密钥（密码形式输入）
4. **模型名称**：要使用的模型名称
5. **请求超时**：API请求的超时时间（秒）

#### 常用模型配置

**华为云ModelArts：**
- Base URL: `https://api.modelarts-maas.com/v2/chat/completions`
- 模型名称: `qwen3-30b-a3b`（通义千问）
- 认证方式: Bearer Token

**百度千帆Coding Plan：**
- Base URL: `https://qianfan.baidubce.com/v2/coding/chat/completions`
- 模型名称: `qianfan-code-latest`
- 认证方式: API Key

**OpenAI：**
- Base URL: `https://api.openai.com/v1/chat/completions`
- 模型名称: `gpt-3.5-turbo` 或 `gpt-4`
- 认证方式: Bearer Token

#### 测试连接

配置完成后，点击 `测试连接` 按钮验证配置：

- ✅ **成功**：弹出"连接成功！"提示
- ❌ **失败**：弹出错误信息，请检查配置

#### 恢复默认配置

点击 `恢复默认配置` 按钮将所有设置重置为默认值。

## 生成报告

### 1. 打开生成界面

在IDEA菜单栏：

- `Tools` -> `生成工作报告`
- 或使用快捷键（如果已配置）

### 2. 选择报告类型

支持以下报告类型：

- **日报**：分析最近一天的工作
- **周报**：分析最近一周的工作
- **月报**：分析最近一个月的工作
- **自定义**：指定自定义时间范围

### 3. 选择时间范围

根据报告类型，选择时间范围：

- **日报**：选择日期（默认今天）
- **周报**：选择周数（默认本周）
- **月报**：选择月份（默认本月）
- **自定义**：选择开始和结束日期

### 4. 生成报告

点击 `生成` 按钮，插件将：

1. 分析所有已启用项目的Git提交记录
2. 收集提交信息（作者、时间、消息等）
3. 调用AI服务生成工作总结
4. 生成Markdown格式的报告
5. 保存到配置的输出目录

### 5. 查看报告

报告生成完成后：

- 自动打开报告文件
- 文件名格式：`工作报告_YYYY-MM-DD.md`
- 位置：配置的输出目录

## 报告格式

生成的报告包含以下部分：

### 1. 基本信息

- 报告类型（日报/周报/月报）
- 时间范围
- 生成时间

### 2. 项目列表

- 项目名称
- 项目路径
- Git仓库状态

### 3. 工作总结

由AI生成的智能总结，包括：
- 主要工作内容
- 重要提交
- 代码变更概述

### 4. 详细提交记录

每个项目的详细提交信息：
- 提交哈希
- 提交时间
- 提交作者
- 提交消息

## AI配置说明

### 获取API密钥

#### 华为云ModelArts

1. 访问：https://console.huaweicloud.com/modelarts/
2. 登录华为云账号
3. 进入"API密钥"管理
4. 创建新的API密钥
5. 复制密钥到插件配置

#### 百度千帆

1. 访问：https://console.bce.baidu.com/qianfan/
2. 登录百度账号
3. 进入"API Key"管理
4. 获取或创建API Key
5. 复制密钥到插件配置

#### OpenAI

1. 访问：https://platform.openai.com/api-keys
2. 登录OpenAI账号
3. 点击"Create new secret key"
4. 复制生成的密钥到插件配置

### 模型选择

选择模型时考虑：

- **性能**：大模型响应更慢但质量更高
- **成本**：不同模型定价不同
- **功能**：某些模型可能不支持特定功能

建议：
- 日常使用：中小型模型（如 `qwen3-30b-a3b`）
- 重要报告：大型模型（如 `gpt-4`）
- 快速测试：小型模型（如 `gpt-3.5-turbo`）

## 常见问题

### Q1: 插件安装后找不到菜单

**A:** 
1. 确认已重启IDEA
2. 检查插件是否启用：`Settings` -> `Plugins`
3. 查看插件版本是否兼容当前IDEA版本

### Q2: 测试连接失败

**A:** 检查以下几点：
1. API Key是否正确
2. Base URL是否正确
3. 网络连接是否正常
4. API服务是否可用
5. 超时时间是否过短

### Q3: 报告生成失败

**A:** 可能原因：
1. 项目不是Git仓库
2. 没有符合条件的提交记录
3. AI服务调用失败
4. 输出路径没有写入权限

### Q4: AI总结不准确

**A:** 尝试：
1. 切换到更强大的模型
2. 调整时间范围，减少提交数量
3. 检查提交消息是否清晰
4. 手动编辑报告内容

### Q5: 编码问题（中文乱码）

**A:** 
1. 确保使用UTF-8编码
2. 检查IDEA文件编码设置
3. 使用支持UTF-8的编辑器查看报告

### Q6: 如何更新插件

**A:**
1. 下载新版本插件JAR
2. 卸载旧版本：`Settings` -> `Plugins` -> 卸载
3. 安装新版本
4. 重启IDEA

### Q7: 多个项目如何管理

**A:** 
1. 添加所有相关项目
2. 使用"启用/禁用"功能控制哪些项目出现在报告中
3. 可以为不同报告类型创建不同的项目组合

### Q8: 报告可以自定义吗

**A:** 当前版本：
- 生成的报告是Markdown格式
- 可以手动编辑报告内容
- 可以修改模板（需要修改源代码）

未来版本计划支持：
- 自定义报告模板
- 自定义报告格式（HTML、PDF等）
